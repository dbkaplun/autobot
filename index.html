Autobot

Self-building websites • Fork on Github

Installation

  • jQuery: <script src="//code.jquery.com/jquery-1.11.3.min.js"></script>
  • HighlightJS (optional): <script src="//cdnjs.cloudflare.com/ajax/libs/highlight.js/8.5/highlight.min.js"></script>
  • Autobot: <script src="//dbkaplun.github.io/autobot/autobot.js"></script>

Usage

Add a data-autobot=name attribute to any elements you want animated. That's it!
